
📌 Project Overview

This project is a UDP-based client-server application developed in Python as part of the Computer Networks 2025/26 course.

The system enables multiple clients to communicate with a server over a real network using the UDP protocol, allowing operations such as reading, writing, and executing commands on server-side files.

🎯 Objectives
Implement a UDP server that listens on a specific IP address and port.
Allow multiple clients (at least 4 devices) to connect and send requests.
Support different access levels (admin vs regular clients).
Enable file operations on the server (read, write, execute).
Ensure proper communication between client and server.
